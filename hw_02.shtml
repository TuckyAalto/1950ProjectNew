  • Create an HTML5 web page

    Build an HTML5 page that describes two or more places you have lived or visited other than Vancouver BC (cities, countries, any locales of your choice). Include text and images to inform the user about each place. At minimum, use one image and several sentences to describe each place. You may use images that you find online, but the textual content should be authored by you. Describe what you liked or did not like about the place, and some useful or interesting information you know (eg: where your favorite restaurant is).

    Additional Criteria

    Using any resources available to you, investigate two new HTML5 elements or features not demonstrated in class. Use these elements on the page. Include on this page the following information about each of the two new HTML tags:

    • Descriptions of the semantics/rules of use (eg: parent/child relationships)
    • A list of relevant attributes and details about how to use each attribute
    • Browser compatibility issues (if any) for this element across IE, Firefox, Chrome, Safari, and/or Opera.
    • Any additional notes you deem relevent
