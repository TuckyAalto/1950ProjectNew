Homework | S01

  • Publish your COMP1950 homepage

    Create a homepage to serve as the hub for all your work in COMP1950. As subsequent homework assignments are issued, you will add links to them on this page. Your homepage should state your full name, or nickname if you prefer. Include some biographical information about yourself (such as from your introduction today) and a photo.

    Also include a brief overview of your skill level, web development work you have done so far (if any), as well as a list of tools, plug-ins, applications or IDEs that you use for web development, including links to the web pages of these tools, and links to some of your favorite sites, accompanied by brief descriptions explaining why these sites are favorites.

    Use your understanding of HTML and CSS to the limit of your knowledge, but ensure that it is usable, presentable and coherent. Use CSS, not HTML tables, to define the layout of this page. Include the use of both .class and #id CSS selectors. Do NOT use any Javascripting or server side scripting. Be prepared to share this page with your colleagues in subsequent homework assignments: design and build a page you can be proud of. This page will be used as the entry point for all of your COMP1950 work, edited each week. Don't make your instructor have to search for your homework.

    Additional Criteria

    This and all subsequent COMP 1950 homework assignments are due the day before the following class unless otherwise noted (eg: if assigned Monday, Jan 1, homework is due Tuesday, Jan 2 11:59pm). Only files located on the COMP 1950 web server will be marked. All submissions must include:

    • Well formed, valid, semantic HTML. Structural errors are not permitted (eg: misplacing a li tag, or duplicating an id assignment). Non-structural errors are permissible (eg: using a color name not recognized by the validator)
    • Include a referer link on all your COMP1950 pages
    • Good file/folder naming conventions
    • Good code practices. (Comment major sections of code, use tabs to indicate tag inheritance)

    List of favourite tools and resources

    Links to some of your favourite sites

    CSS stylesheet

    Use of both .class and #id selectors

Homework | S02

  • Create an HTML5 web page

    Build an HTML5 page that describes two or more places you have lived or visited other than Vancouver BC (cities, countries, any locales of your choice). Include text and images to inform the user about each place. At minimum, use one image and several sentences to describe each place. You may use images that you find online, but the textual content should be authored by you. Describe what you liked or did not like about the place, and some useful or interesting information you know (eg: where your favorite restaurant is).

    Additional Criteria

    Using any resources available to you, investigate two new HTML5 elements or features not demonstrated in class. Use these elements on the page. Include on this page the following information about each of the two new HTML tags:

    • Descriptions of the semantics/rules of use (eg: parent/child relationships)
    • A list of relevant attributes and details about how to use each attribute
    • Browser compatibility issues (if any) for this element across IE, Firefox, Chrome, Safari, and/or Opera.
    • Any additional notes you deem relevent
    Download the PDF

Homework | S03

  • Use advanced CSS techniques and alternate stylesheets

    You will need to partner and coordinate with another COMP1950 student to complete this lab.

    Create an alternate stylesheet for the homework # 1 homepage of your partner. Since you won’t have access to their FTP credentials, you will need to have them provide you with a copy of all relevant files (html, css, images, etc). Once you have created your new stylesheet send your partner a copy (and any other relevant files such as background images your CSS uses).

    As much as possible, try to make your alternate stylesheet display your partner’s homepage as significantly different. Your stylesheet must style the page with different fonts/typesetting, color schemes, and possibly also layouts and background images. Make an effort to create a visually appealing style that is clearly distinct from the original.

    Do not make changes to the HTML code of your partner’s homepage, other than adding the appropriate link tags to the head section. When you receive the styles your partner writes for your homepage, apply them as an alternate stylesheet. The title of your alternate stylesheet MUST include the fullname of your partner.

    Additional Criteria

    Demonstrate the following CSS techniques in your stylesheet:

    • id, class and HTML selectors
    • Descendant selectors
    • Child Selectors
    • Attribute selectors
    • Pseudo-classes
