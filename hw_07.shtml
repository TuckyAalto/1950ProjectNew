  • Deploy a jQuery plugin

    Write an XHTML1.0 or HTML5 page. Add a link to this page on your homepage. Search online for a jQuery plugin that you find useful, and that we did not cover in class. Learn how to use it, and deploy it in this page. Demonstrate attempts to alter the style or behavior of the plugin to suit your design (change the color, sizes, speed of animation, etc).

    If you cannot find any plugins you like, consult the list of links in the Plugin section of session seven online lecture notes: http://bcitcomp.com/comp1950/lectures/seven/. If you have trouble implementing the plugin of your choice, consider finding a different one that you can work with more easily.

    Additional Criteria

    Include the following:

    • Use the plugin (eg if you chose a navigation plugin, demonstrate it by displaying the navigation in your page)
    • A text description of the plugin (in your own words)
    • A list of any style/behavior you have customized
    • A link to the site you downloaded the plugin from
    • A link(s) to the page(s) that taught you how to use it
